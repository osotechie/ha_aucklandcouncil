# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this integration, please report it responsibly using **GitHub's private vulnerability reporting**:

1. Go to the **Security** tab of this repository
2. Click **"Report a vulnerability"**
3. Fill in the details of the vulnerability

Please **do not** open a public issue for security vulnerabilities.

## What to Expect

- Acknowledgement within **72 hours**
- An assessment and initial response within **7 days**
- We will work with you privately to understand and resolve the issue before any public disclosure
